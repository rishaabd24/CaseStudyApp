# CaseStudy

## Technologies

C# ASP.NET core 5 (MVC ) , 3-tier Architechture, EF Code First
Authorization using Razor Pages


## Features
- Registration and Login
- Upload files (Restricted to only .PBIX Files)
- Manage user details
- View File Metadata and Download
